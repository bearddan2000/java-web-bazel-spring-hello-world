# java-web-bazel-spring-hello-world

## Description
This is a POC project to demonstrate the
bazel build tool that uses
spring boot as a tratransitive dependency.

## Tech stack
- bazel
- java
    - spring

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
